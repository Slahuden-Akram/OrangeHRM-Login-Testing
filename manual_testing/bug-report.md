# Bug Report

## Bug ID
BUG-001

## Title
Forgot Password request keeps loading after entering username

## Module
Login / Forgot Password

## Test Environment
- Website: OrangeHRM Demo
- Browser: Chrome
- OS: Windows

## Steps to Reproduce
1. Open the OrangeHRM demo login page.
2. Click on “Forgot your password?”
3. Enter username: Admin
4. Click on “Reset Password”.

## Actual Result
The application keeps buffering/loading and no confirmation message is displayed.

## Expected Result
The system should display a clear confirmation or error message after clicking “Reset Password”.

## Severity
Medium

## Priority
Medium

## Status
Open

## Notes
Issue reproduced more than once during manual testing.
